A modern, responsive, and production-ready **blog website** built using **Bootstrap 5** and enhanced with custom styling, full-page routing, and interactive features. ##  Core Features

-  Responsive navigation bar with **search functionality** and mobile toggle
-  Eye-catching **hero section** with a purple-blue gradient background
-  **Grid-based layout** featuring 6 fully developed blog post previews
-  **Newsletter subscription** section with input validation
-  Comprehensive **footer** with social media icons and link groups

---

##  Design Elements

-  **Bootstrap 5 CDN** for rapid development and responsiveness
-  Custom CSS for gradients, transitions, hover effects & animations
-  **Lucide-react** icons for sleek social icons and visuals
-  Responsive design working across desktops, tablets, and mobile phones
-  Modern cards with:
  - Blog title & thumbnail
  - Category tags
  - Author avatar & name
  - Read time
  - Call-to-action buttons

##  Functionality Additions

###  Complete Routing System

-  **Home Page** with search bar
-  **Individual Blog Post Pages** with full-length articles (~2000+ words each)
-  **Categories Page** with tag-based filtering
-  **About Page** featuring team info
-  **Contact Page** with functional form and validation

---

###  Real Blog Content

Each blog post includes:

-  Fully written articles (~2000+ words)
-  HTML elements: headings, code snippets, blockquotes, and lists
-  Related posts section
-  Tag-based categorization
-  SEO-optimized URL slugs

##  Interactive Features

-  **Load More** functionality for progressive content loading
-  **Responsive Navigation** with active link highlighting
-  **Contact Form** with validation and structured submission
